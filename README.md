# newspaper-scraping

simple newspaper scraping example with [newspaper3k](https://newspaper.readthedocs.io) and [Natural Language Tool Kit](https://www.nltk.org/)

## install

```
pip install -r requirements.txt
```

## use

```
python scrap.py
```
